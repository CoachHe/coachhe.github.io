# 对象的实例化

## 创建对象的方式
- new
   - 最常见的方式
   - 变形1：Xxx的静态方法
   - 变形2：Xxx Builder/Factory
- Class的newInstance()：反射的方法，只能调用空参构造器，权限必须是public
- Constructor的newInstance(Xxx)：反射的方式，可以调用无参或带参构造器，权限不需要是public
- 使用clone()
- 使用反序列化
- 第三方库Objenesis
